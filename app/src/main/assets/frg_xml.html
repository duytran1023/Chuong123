Fragments in XML

Bạn có thể nghĩ về fragment như một phần mô-đun của activity, có vòng lặp riêng, nhận các sự kiện đầu vào của riêng nó và bạn có thể thêm hoặc xóa trong khi activity đang chạy (giống như một "sub activity" mà bạn có thể tái sử dụng trong activity khác nhau) Nói một cách đơn giản, chúng ta có thể nói "Fragment là một thành phần UI có thể sử dụng lại". Nó có vòng đời riêng và xử lý các sự kiện của riêng mình. Nó có thể được thêm hoặc loại bỏ trong khi activity chạy.

Công dụng của Fragments

  1. Giao diện người dùng linh hoạt trên các kích thước màn hình khác nhau
  2. Hiển thị tab Fixed/ Scrolling/ Swipe
  3. Hộp thoại Dialog
  4. Tùy chỉnh thanh điều khiển với các chế độ danh sách và tab

Các bước để tạo Fragments trong XML Layout:

  1. Tạo class extends Fragment
    2. public class MyFragment extends Fragment

  2. Tạo một tệp l hoặc các đối tượng java riêng biệt có chứa giao diện
    3. <FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
                android:layout_width="match_parent"
                android:layout_height="match_parent"
                xmlns:tools="http://schemas.android.com/tools"
                tools:context="demo.arjun.com.fragmentapp.MyFragment"
                android:id="@+id/cdcd12121">
                
                <!--TODO: Update blank fragment layout -->
                <TextView
                    android:layout_width="match_parent"
                    android:layout_height="match_parent"
                    android:text="@string/hello_blank_fragment">
                </TextView>
            </FrameLayout>

  3. Sử dụng phương thức OnCreate để liên kết appearance
    4. public class MyFragment extends Fragment {
                public View onCreateView(LayoutInflater inflater, ViewGroup container, Bundle savedInstanceState) {
                //Inflate the layout for this fragment
                return inflater.inflate(R.layout.fragment_my, container, false);
                    }
                }

  4. Đính kèm Fragment vào trong Activity qua XML
    5. <fragment android:layout_width="match_parent"
                android:layout_height="match_parent"   android:name="demo.arjun.com.fragmentapp.MyFragment"
                android:id="@+id/myfragment">
            </fragment>
