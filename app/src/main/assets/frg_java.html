Fragments in Java

Bạn nên tạo đoạn trong Java chứ không phải bằng XML, vì chúng ta có thể thay đổi UI trong thời gian chạy nếu chúng ta tạo fragment trong Java.

Fragment Manager

FragmentManager quản lý Fragments trong Android, cụ thể là nó xử lý các transactions giữa các fragments.
Một transactions là một cách để thêm, thay thế hoặc loại bỏ fragments.

Mỗi hoạt động đều có Fragment Manager riêng có thể truy cập thông qua getFragmentManager().
Nó duy trì các tham chiếu đến tất cả các fragments bên trong Activity.
Sử dụng findFragmentById () hoặc findFigatenetByTag () để tham chiếu đến một Fragment cụ thể.

Fragment Transaction

Có thể quản lý các thay đổi đối với UI theo cách thêm, xóa và thay thế fragment.

Các bước quản lý fragment transaction:

  1. Begin transaction
  2. Add, remove, replace fragment mong muốn
  3. Commit transaction

Các bước tạo Fragment trong Java:

  1. Tạo một class extents fragment. 
    public class MyFragment1 extends Fragment

  2. Tạo một file java riêng hoặc các đối tượng java có chứa appearance 
    <FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
            android:layout_width="match_parent"
            android:layout_height="match_parent"
            xmlns:tools="http://schemas.android.com/tools"
            tools:context="demo.arjun.com.fragmentapp.MyFragment"
            android:id="@+id/cdcd12121">
            
            <!--TODO: Update blank fragment layout -->
            <TextView
                android:layout_width="match_parent"
                android:layout_height="match_parent"
                android:text="@string/hello_blank_fragment">
            </TextView>
        </FrameLayout>

  3. Sử dụng phương thức onCreateView liên kết appearance 
    public class MyFragment extends Fragment {
                public View onCreateView(LayoutInflater inflater, ViewGroup container,
                Bundle savedInstanceState) {
                //Inflate the layout for this fragment
                return inflater.inflate(R.layout.fragment_my, container, false);
                    }
                }

  4. Đính kèm fragment bên trong activity thông qua java 
    //Create an object of your fragment here it is frag
                MyFragment frag = new MyFragment();
                FragmentManager manager = getFragmentManager();
                FragmentTransaction transaction = manager.beginTransaction();
                transaction.add(R.id.my_fragmentlayout_name, frag, "frag_tag");
                transaction.commit();
