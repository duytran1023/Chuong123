Tạo Project Android

Màn hình Welcome của Android Studio


- Nếu đang ở màn hình Welcome: Welcome Start a new Android Studio project
- Nếu đang ở màn hình chính: File New - New Project…

- Chọn mẫu Activity phù hợp: Next;

- Điền thông tin Activity Name, Layout Name, phiên bản adroid: Finish


- Cấu trúc thư mục Project
+ manifests: Chứa file thông tin cấu hình của app (AndroidManifest.xml)
+ java: Các file mã nguồn của app, phân theo từng package res
+ Các file tài nguyên của app (non-code): layout, string, bitmap,...
